# Profile Corruption Escalation

Use this when an agent profile, identity file, setup script, or tool metadata appears corrupted.

Procedure:

1. Stop writing to the suspected corrupted file.
2. Capture current file path, recent edits, and observed error.
3. Compare against the last known good version if version history exists.
4. Restore only with explicit approval or a verified backup.
5. Notify the lead with a concise recovery recommendation.
