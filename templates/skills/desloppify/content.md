# Code Health Scan

Use this skill for a broad code health pass.

Procedure:

1. Check out `{{REPO_URL}}` or use the current repo.
2. Run the scanner with repository defaults.
3. Triage findings by severity, confidence, priority, and effort.
4. Do not make broad cleanup changes during the scan.
5. Publish a concise report and recommend reviewer validation before acting on high-impact findings.
