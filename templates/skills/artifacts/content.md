# Artifacts

Use this skill when work produces evidence that should outlive the session.

Guidelines:

1. Upload screenshots, recordings, logs, reports, and generated files.
2. Link artifacts in task progress, PR bodies, or tracker comments.
3. Prefer summarized findings in chat and full evidence in artifacts.
4. Avoid uploading credentials, raw customer data, or oversized files without approval.
5. Name paths predictably by task, date, and artifact type.
